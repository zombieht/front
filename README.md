# fun
    use jhipster
## Development
1.安装node v6.9+

2.设置npm镜像仓库 
```shell
npm config set registry https://registry.npm.taobao.org 
```
3.安装
```shell
npm install -g yarn
```
4.安装 jhipster（选装 用于代码生成）
```shell
npm install -g yo
npm install -g generator-jhipster
```
5.clone 项目到本地，运行安装依赖包
```shell
yarn install
```
6.启动项目 
```shell
yarn start
```
